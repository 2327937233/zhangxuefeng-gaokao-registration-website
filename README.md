# -
本项目采用前后端分离的技术思路，前端使用原生 HTML、CSS 和 JavaScript 构建单页交互界面，后端基于 Python FastAPI 搭建 API 服务，并通过 Pydantic 完成数据建模与参数校验。在智能分析层面，系统接入 DeepSeek 大模型，结合本地高校专业知识库、规则推荐引擎、意图识别、联网检索、Prompt Engineering 和回答守卫机制，实现志愿咨询、冲稳保推荐、专业就业分析和 10 年后压力测试等功能。同时，项目引入“张雪峰式志愿咨询 Skill”，将教育选择、家庭风险、专业壁垒、城市资源和就业回报等分析框架沉淀为可复用的咨询能力，使系统不仅能生成回答，还能按照特定专家风格和决策逻辑给出更贴近真实志愿填报场景的建议。
